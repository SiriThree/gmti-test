# 🧙‍♂️ GMTI-Test：诡秘之主途径测试
> 一个基于 GitHub Pages 的沉浸式《诡秘之主》人格与途径测试网站

### 🔗 直接体验
**立即进入网站测试你的神性途径：**
https://sirithree.github.io/gmti-test/

### 📖 项目介绍
这是一个专为《诡秘之主》粉丝定制的心理测试网站。
通过 25 道精心设计的题目，你可以测出你最契合的**神性途径**，并获得一段专属的、充满诡秘气质的详细解析！

### ✨ 网站特色
- 🎨 **视觉盛宴**：精心设计的暗黑赛博风界面，配合粒子动效
- 📝 **完善题库**：剔除尴尬问题，全部替换为贴合大学生与诡秘世界观的题目
- 🧠 **精准匹配**：基于你选择的逻辑，计算出最符合你本性的途径
- 📱 **移动适配**：手机、平板、电脑全部完美适配

### 🛠 技术栈
- **前端框架**：HTML5 + Tailwind CSS v3
- **部署平台**：GitHub Pages
- **动画效果**：tsParticles (粒子背景)
- **交互逻辑**：原生 JavaScript
